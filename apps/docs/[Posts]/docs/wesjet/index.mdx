This tutorial will get you comfortable with the basics of Wesjet by walking through how to build a simple blog site using Next.js.

The blog will include a list of posts with a title, body, and date, rendered with the most recent on top. And each post will have its own page.

If you'd like to dive right in and start playing with Wesjet, you can open the project in [Gitpod](https://gitpod.io).
